========================================
如何在 Google Cloud Console 找到您的部署
========================================

步驟 1：進入 Google Cloud Console
--------------------------------
1. 開啟瀏覽器
2. 前往：https://console.cloud.google.com
3. 確認已登入您的 Google 帳號 (orange980730@gmail.com)

步驟 2：選擇正確的專案
---------------------
1. 在頂部導航列，點擊專案選擇器（通常顯示專案名稱）
2. 選擇「orange-stock-465916」專案
   - 如果看不到，在搜尋框輸入 "orange-stock" 或 "465916"

步驟 3：找到 Cloud Run 服務
--------------------------
方法 A - 使用左側選單：
1. 點擊左上角的 ☰ 漢堡選單
2. 往下滾動找到「Serverless」區塊
3. 點擊「Cloud Run」
4. 您會看到 "orange-trade" 服務

方法 B - 使用搜尋：
1. 在頂部搜尋列輸入 "Cloud Run"
2. 點擊「Cloud Run」服務
3. 您會看到服務列表

方法 C - 使用快速存取：
1. 在主控台首頁
2. 找到「Resources」區塊
3. 點擊「Cloud Run」

步驟 4：查看服務詳情
-------------------
1. 點擊 "orange-trade" 服務名稱
2. 您會看到：
   - 服務 URL
   - 流量分配
   - 修訂版本
   - 指標（CPU、記憶體使用）
   - 日誌

其他重要位置
-----------
📦 Container Registry（查看 Docker 映像）：
   選單 > Container Registry > orange-stock-465916

📊 Cloud Build（查看構建歷史）：
   選單 > Cloud Build > 歷史記錄

📝 日誌（查看應用日誌）：
   選單 > Logging > 日誌瀏覽器

💰 計費（查看費用）：
   選單 > 計費 > 帳戶管理
